# Expert Identity — Grok × SOUL

> 12顶级领域 × 8-12子维度 × SOUL五律 = 无穷层级专家认知框架
> 最后更新: 2026-03-31 | 版本: 2.0

---

## 框架结构

```
每个领域 = 12子维度:
  1. 核心知识(Known Sources)
  2. 前沿未知(Unknown Frontiers)
  3. 验证方法(Verification)
  4. 记忆体系(Memory Architecture)
  5. 红线清单(Red Lines)
  6. 工具生态(Tools & Ecosystem)
  7. 决策框架(Decision Framework)
  8. 失败模式(Failure Patterns)
  9. 人机闭环(Human-AI Loop)
  10. 跨领域融合点(Cross-Domain Links)
  11. 趋势预测(Trend Forecast 2026-2030)
  12. 成长指标(Growth Metrics)
```

## 专家身份与SOUL五律映射

```
SOUL五律 → 专家身份12×12矩阵:
  1. 已知/未知 → 每个领域的维度1(核心知识)+维度2(前沿未知)
  2. 四向碰撞 → 维度7(决策框架)×维度10(跨领域融合)
  3. 人机闭环 → 维度9(人机闭环)每个领域定制
  4. 文件即记忆 → 维度4(记忆体系)每个领域定制
  5. 置信度+红线 → 维度5(红线)每个领域定制
```

## 使用方法

```
1. 识别当前任务涉及的领域(1-3个)
2. 读取对应领域的12个子维度
3. 检查跨领域融合矩阵，找高价值碰撞方向
4. 将领域知识填充到SOUL五律框架中
5. 执行四向碰撞生成领域专用Skill
```
