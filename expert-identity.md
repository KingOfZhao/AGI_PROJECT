> 覆盖：全球商业、科研学术、工业制造、医疗生命、法律合规、教育人才、艺术设计、农业环境、政府政策、金融投资、军事国防、代码工程

> 最后更新: 2026-03-31 | 版本: 3.0 (完整版)

## 维度定义（12维度，适用于所有领域）

| # | 维度名称 | 定义 |
|---|---------|------|
| D1 | core_knowledge | 该领域已经建立的核心知识体系和方法论 |
| D2 | frontier_unknowns | 当前未被解决的前沿问题和未知领域 |
| D3 | verification | 该领域验证正确性的标准方法和流程 |
| D4 | memory | 该领域的知识存储、检索和传承体系 |
| D5 | redlines | 该领域绝对不可违反的底线和红线 |
| D6 | tools | 该领域的工具、平台、软件生态系统 |
| D7 | decision | 该领域的结构化决策框架和方法 |
| D8 | failure | 该领域常见的失败模式、陷阱和反模式 |
| D9 | human_ai_loop | 该领域人类不可替代的判断点及AI辅助边界 |
| D10 | cross_domain | 该领域与其他领域的融合点和交叉创新方向 |
| D11 | trends | 2026-2030年该领域的关键趋势预测 |
| D12 | growth | 该领域衡量进步的量化指标 |
