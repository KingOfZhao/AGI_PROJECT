## 解决什么问题？

AI读论文只能总结"方法X达到精度Y"，不能判断可复现性、发现隐藏机会、管理假设全生命周期。
`researcher-cognition` 把SOUL五律转化为科研实践：

- **已知/未知** → 假设驱动的研究方法
- **四向碰撞** → 文献碰撞四向推理（正面复现/反面局限/侧面附录/整体趋势）
- **人机闭环** → AI文献碰撞 → 人类实验设计 → AI数据分析 → 循环
- **文件即记忆** → literature_review/ + experiments/ + insights/ 结构化研究记忆
- **置信度+红线** → 6条科研红线 + 可复现性要求

## 核心特色

- 📚 文献碰撞四向推理（正面复现/反面局限/侧面附录/整体趋势）
- 🧪 假设全生命周期管理（提出→设计实验→验证/证伪）
- 📁 结构化研究记忆（可复现实验记录）
- 🔴 6条科研红线（不伪造数据、不cherry-pick、不忽略矛盾...）
