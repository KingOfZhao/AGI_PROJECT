## 解决什么问题？

工厂之前只做3步（生成→验证→发布），Skill发布后就没人管了。
`skill-factory-optimizer` 补齐后3步：

- 📢 **推广引擎**: 每个新Skill自动产出知乎/CSDN/X/Discord推广文（四向碰撞写法）
- 📊 **质量监控**: 8+指标实时监控，低置信度/孤立Skill/推广积压自动告警
- 🔄 **工厂自进化**: RAPVL循环应用于工厂自身，Growth Score驱动碰撞参数优化

## 工厂全闭环

```
之前: 生成 → 验证 → 发布 （3步）
现在: 生成 → 验证 → 发布 → 推广 → 监控 → 优化 （6步闭环）
```
