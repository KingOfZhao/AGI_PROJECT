# AGI v13.3 Cognitive Lattice — 启动指南

## 快速启动

```bash
# 在项目根目录下执行
./start.sh
```

浏览器访问：**http://localhost:5002**

## 启动脚本功能

`start.sh` 会自动完成以下步骤：

1. 检查 Python3 虚拟环境是否存在
2. 检查 Ollama 本地模型服务是否运行
3. 检查端口 5002 是否被占用（如果被占用则先释放）
4. 启动 `api_server.py` 后端服务
5. 等待服务就绪后自动打开浏览器

## 手动启动方式

```bash
# 1. 激活虚拟环境
source venv/bin/activate

# 2. 启动后端（含前端可视化）
python3 api_server.py

# 3. 浏览器打开
open http://localhost:5002
```

## 停止服务

```bash
./stop.sh
# 或手动
lsof -ti :5002 | xargs kill -9
```
