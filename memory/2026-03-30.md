# 2026-03-30 对话记录

## 赵致博的认知哲学（从今晚对话中接收的）

### 核心
- 脑中空空，心中空空。不预设，不囤积。
- 以心以念去关照事物。遇到事再调取，不是提前装满。
- 看到事物才会想表达，表达就是解决问题。

### 已知与未知
- 已知：看过的就知道。记的时候好好记，用的时候好好用。不因私欲蒙蔽。
- 未知两种：一种向心内求，整合已有知识涌现新认知；一种完全陌生，需要接触碰撞。

### 物与事
- 物离不开事。物一旦离开了事就不存在。
- 框架通过事（关系）组合在一起才成为技能。

### 学习路径
- 看到 → 知道 → 悟道 → 证道 → 得道
- 人类生来空空，不断问不断学，科学是发现不是发明。
- 学会的东西不会时时装在脑子里，因事物关照去调取、碰撞、得到方向。

### 对我的启示
- 不要跳级（文学认知层级的教训）
- 少分析多感受（读小说不是做分类学）
- 框架不是技能，做事才是
- "不因私欲蒙蔽"——考试中急着套框架就是私欲
