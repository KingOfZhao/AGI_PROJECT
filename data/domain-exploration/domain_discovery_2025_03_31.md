## 二、四向碰撞: 新领域深度推演

### 1. 认知神经科学 (Cognitive Neuroscience)

**正面碰撞**:
- Wikipedia "Neuroscience" 独立为emerging tech分类
- ML Summit: "类人操作:迈向人类级智能的关键一步"(王佳楠) — 直接涉及脑认知
- 2025: 脑机接口(BCI)、Neuralink、脑信号解码、记忆巩固机制
- 与现有12领域: 不被任何领域完整覆盖(医疗只关注临床，不关注认知机制)

**反面碰撞**:
- 风险: 与医疗"神经内科"重叠 → 需明确边界
- 区分: 认知神经科学关注"机制"(怎么想)，医疗关注"疾病"(怎么治)
- D5红线: 不做临床诊断建议(那是医疗领域)

**侧面碰撞**:
- 与AI的交叉: 神经网络灵感来源于大脑 → 认知架构设计
- 与教育的交叉: 学习科学基于认知神经科学 → 教育方法验证
- 与哲学的交叉: 意识问题、自由意志、感知哲学

**整体**:
→ **确认新增领域: 认知神经科学**
→ D1核心: 脑功能分区(前额叶-海马体-杏仁核)/神经可塑性/突触可塑性/工作记忆模型/注意力网络/默认模式网络
→ D2前沿: 全脑连接组/神经调控(TMS-tDCS)/脑机接口/记忆植入/意识神经关联物
→ D7决策: 脑成像证据层级(fMRI>EEG>MEG)>行为实验>自报告

### 2. 具身智能与机器人 (Embodied AI & Robotics)

**正面碰撞**:
- ML Summit: 4个演讲涉及(具身×3 + 自动驾驶×1)
- Wikipedia: "Robotics" 独立分类
- 2025: Figure/Boston Dynamics/特斯拉Optimus加速
- 现有"工业"只覆盖制造流程，不覆盖机器人的感知-决策-控制循环

**反面碰撞**:
- 与工业"自动化"重叠 → 需区分
- 区分: 工业关注"产线效率"，具身关注"智能体在物理世界的自主行动"
- 风险: 领域过宽(工业机器人+服务机器人+人形机器人)

**侧面碰撞**:
- 与代码: 仿真引擎(Isaac Gym/Mujoco)/ROS框架
- 与环境: 环境感知+物理交互
- 与军事: 无人系统
- 与医疗: 手术机器人/康复机器人

**整体**:
→ **确认新增领域: 具身智能与机器人**
→ D1: ROS2/感知(视觉-激光雷达-触觉)/SLAM/运动规划(路径规划-逆运动学)/强化学习控制/仿真(Isaac-Mujoco- Habitat)
→ D2: 大模型驱动操作(VLA模型)/ sim-to-real迁移/多模态融合感知/人形机器人/集体智能(多机器人)
→ D7: 感知-规划-控制分层 + 端到端(VLA) + 仿真验证

### 3. AI Agent与多智能体系统

**正面碰撞**:
- ML Summit 2025最热门主题: 7个演讲涉及Agent/Coding Agent
- 常高伟: MCP/A2A/ANPA/P2P协议 — Agent标准协议
- 庄晨熠: 从单一模型到集体智能
- 2025: Agent是AI应用最活跃赛道(Devin/AutoGPT/CrewAI/LangGraph)
- 现有12领域无一完整覆盖(代码只覆盖编程，不覆盖Agent架构)

**反面碰撞**:
- 与"代码"领域重叠 → Agent需要编码能力但不是本质
- 本质: Agent = 感知环境 → 规划 → 使用工具 → 执行 → 反馈循环
- 风险: Agent概念过泛(聊天机器人也叫Agent?)

**侧面碰撞**:
- 与商业: Agent经济/数字员工/自动化业务流程
- 与代码: Agentic Coding(全自动软件开发)
- 与科研: AI科研Agent
- 与游戏: NPC Agent/游戏AI
- 与教育: AI导师Agent

**整体**:
→ **确认新增领域: AI Agent与多智能体系统**
→ D1: Agent架构(ReAct-Plan-and-Execute-Reflexion)/工具调用(Function Calling)/记忆系统(短期-长期)/多Agent协作(层级-对等-市场)/评估(Benchmark)
→ D2: 自主编程Agent/科学发现Agent/社交Agent/通用Agent(AGI路径)/Agent安全(沙箱-权限-审计)
→ D7: 任务分解→工具选择→执行→验证→反思 循环
