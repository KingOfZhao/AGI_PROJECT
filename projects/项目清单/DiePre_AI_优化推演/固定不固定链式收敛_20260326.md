## 一、人类原话 (逐字记录)

> 我们的目标是固定的，数学公式，物理定律也是固定的，刀模的用处是变化的。
> 但是刀模所加工的材质选择以后就会产生新的固定的，比如你选择卡纸那卡纸的折痕加工方式最终呈现的误差和承重都是有最大最小范围的，瓦楞的误差或许比较大但他的三维呈现的承重范围大小与卡纸是不同的。
> 因此固定的会产生不固定的再从不固定的找到固定的再找到不固定找到最大最小范围。
> 转换成二维是固定的最终三维呈现的追求是固定的，我们推演的是如何运用这些固定的，调整不固定的去获得更好的用户体验，
> 从二维图纸到刀模生产再到刀模切割压制原材料，获得二维包装盒，再将二维包装盒粘贴定型让其成为一个具备自身材质结构等等限制下的称重量的三维呈现。

## 二、形式化: 链式收敛模型

### 2.1 核心公理

```
公理1: 目标是固定的 (用户要一个能承重X kg的3D盒子)
公理2: 数学公式、物理定律是固定的 (不随人的意志改变)
公理3: 刀模用途是变化的 (同一把刀可切不同材料、不同形状)
公理4: 材料选择后产生新的固定 (卡纸的误差范围[εmin, εmax]和承重范围[Fmin, Fmax]是确定的)
公理5: 不同材料的"新固定"不同 (卡纸≠瓦楞≠微瓦)
公理6: 2D展开图是固定的 (数学精确)
公理7: 3D呈现目标是固定的 (客户需求确定)
```

### 2.2 链式收敛方程

```
F₀ (物理定律/数学公式)         — 绝对固定
  ↓ 应用于
V₁ (刀模用途选择)              — 变化的
  ↓ 约束于
F₁ (材料选择 → 误差范围/承重范围)  — 新的固定 [εmin, εmax] × [Fmin, Fmax]
  ↓ 组合产生
V₂ (结构设计选择)              — 变化的 (同材料可设计不同结构)
  ↓ 约束于
F₂ (结构力学 → 特定结构的承重)   — 新的固定 (此结构+此材料=确定的承重)
  ↓ 组合产生
V₃ (工艺参数选择)              — 变化的 (压痕深度/粘合方式/...)
  ↓ 约束于
F₃ (工艺极限 → 成品质量范围)    — 新的固定 [质量min, 质量max]
  ↓ 最终
F_goal (3D成品满足目标)         — 固定目标
```

**简记**: F₀ → V₁ → F₁ → V₂ → F₂ → V₃ → F₃ → F_goal

每一次 **V→F 转换** 都是一次"从不固定中找到固定"的过程。
每一次 **F→V 转换** 都是一次"固定约束下的自由度展开"。

### 2.3 全链路映射

```
阶段          | 类型 | 内容                    | 固定/变化
──────────────|──────|─────────────────────────|──────────
1. 数学/物理  | F₀   | E=σ/ε, M=EIκ, F=ma     | 绝对固定
2. 2D图纸设计 | F₀'  | 几何精确的展开图          | 固定(数学精确)
3. 刀模制造   | V₁   | 选择刀型/桥位/排列        | 变化(选择空间)
4. 材料选择   | F₁   | 卡纸→ε∈[0.1,0.3]mm      | 选择后固定
              |      | 瓦楞→ε∈[0.5,1.5]mm      |
              |      | 卡纸→F∈[3,8]kg          |
              |      | 瓦楞→F∈[15,50]kg        |
5. 切割压制   | V₂   | 压力/速度/压痕深度        | 变化(工艺参数)
6. 2D盒片     | F₂   | 切割后的平面件            | 固定(已成型)
7. 折叠粘贴   | V₃   | 粘合方式/位置/用量        | 变化(最后调整)
8. 3D成品     | F₃   | 具备承重能力的3D盒子      | 固定(最终结果)
9. 目标验证   | F_goal| 承重≥X, 尺寸±Y           | 固定(客户需求)
```

## 三、每个阶段的 [min, max] 约束传播

### 3.1 材料选择后的约束锁定 (F₁)

```
材料: 白卡纸 300gsm
├── 厚度: 0.38±0.02 mm → t ∈ [0.36, 0.40]
├── 折痕误差: ε_crease ∈ [0.1, 0.3] mm
├── 弯折半径: R_min = 2t = 0.76 mm (固定)
├── 纤维断裂应变: ε_fracture = 0.035 (固定)
├── 弹性模量: E ∈ [3500, 5500] MPa (纤维方向依赖)
│   ├── MD (纵向): E_MD ∈ [4500, 5500]
│   └── CD (横向): E_CD ∈ [3500, 4200]
├── 回弹角: θ_spring = f(E, t, R, 压痕深度) → 可计算范围
├── 耐破强度: ≥250 kPa (固定下限)
├── 含水率膨胀: 0.1%/1%RH (固定系数)
└── 承重(垂直压溃): F_crush ∈ [3, 8] kg/cm² (取决于楞型)

材料: E瓦楞
├── 厚度: 1.5±0.15 mm → t ∈ [1.35, 1.65]
├── 折痕误差: ε_crease ∈ [0.5, 1.5] mm (比卡纸大5倍)
├── 弯折半径: R_min = 2t = 3.0 mm (比卡纸大4倍)
├── 弹性模量: E ∈ [800, 2000] MPa (比卡纸低)
├── 回弹角: 更大(因为芯纸弹性)
├── 耐破强度: ≥600 kPa (比卡纸高)
├── 含水率膨胀: 0.3%/1%RH (比卡纸高3倍)
└── 承重(垂直压溃): F_crush ∈ [15, 50] kg/cm² (远大于卡纸)
```

**关键洞察**: 选择材料 = 锁定一组[min, max]区间。
不同材料锁定**不同的**区间 → 这就是"从不固定中产生新的固定"。

### 3.2 结构选择后的约束锁定 (F₂)

```
选定: 白卡纸 + 120370托盘盒 + L=300 W=200 H=50

→ 壁面数: 固定 = 5主面+8折耳+4斜角 = 17片 (结构决定)
→ 折线数: 固定 = 16条
→ 每条折线的角度: 固定 (底壁=90°, 折耳=180°, 斜角=135°)
→ 展开面积: 固定 = (300+100+40)×(200+100+40) = 440×340 = 149,600 mm²
→ 材料用量: 固定 = 149,600 × 0.38mm × 0.72g/cm³ = 40.9g
→ 承重(堆叠):
    顶面: L×W = 60,000 mm², F_crush × 面积 × 安全系数
    侧面: 取决于壁高H和材料刚度
    → F_stack ∈ [18, 48] kg (可计算范围)
```

### 3.3 工艺选择后的约束锁定 (F₃)

```
选定: 平压模切 + 胶水粘合 + 自动糊盒机

→ 模切精度: ±0.15mm (机器固定)
→ 压痕精度: ±0.1mm (机器固定)  
→ 压痕深度: 选择后固定 → 决定回弹角
    深度=0.7t → 回弹≈2° → 最终角度=88°
    深度=0.5t → 回弹≈5° → 最终角度=85°
→ 粘合强度: 取决于胶水类型+粘口面积
    热熔胶: 0.8-1.2 MPa
    水性胶: 0.3-0.6 MPa
→ 糊盒速度: 决定胶水固化时间 → 影响粘合强度
    → 速度↑ → 固化不足 → 强度↓
    → 速度↓ → 成本↑

→ 3D成品综合指标:
    尺寸精度: ±(0.15+0.1+ε_crease) = ±0.55mm (白卡) / ±1.75mm (瓦楞)
    承重: F_stack × 粘合强度系数 × 回弹修正
    外观: 折痕清晰度 × 印刷精度 × 表面处理
```

## 四、约束传播计算引擎

### 4.1 数据结构

```python
class FixedConstraint:
    """固定约束 — 选择/确定后不可改变"""
    name: str           # 'E_modulus_MD'
    value_min: float    # 4500
    value_max: float    # 5500
    unit: str           # 'MPa'
    source: str         # 'material_datasheet' / 'physics_law' / 'machine_spec'
    confidence: float   # 0.95
    
class VariableChoice:
    """变化选择 — 用户/系统可调整"""
    name: str           # 'crease_depth_ratio'
    current: float      # 0.7
    range_min: float    # 0.3
    range_max: float    # 0.9
    affects: list[str]  # ['springback_angle', 'fold_clarity', 'crack_risk']

class ConstraintChain:
    """固定→不固定→再固定 的一条链"""
    stages: list[ConstraintStage]
    
class ConstraintStage:
    stage_name: str     # '材料选择'
    stage_type: str     # 'F' or 'V'
    fixed: list[FixedConstraint]
    variables: list[VariableChoice]
    propagates_to: list[str]  # 下游影响
```

### 4.2 约束传播算法

```python
class ConstraintPropagationEngine:
    """从F₀出发, 沿链传播约束, 在每个V节点展开选择空间, 在每个F节点锁定范围"""
    
    def propagate(self, chain: ConstraintChain):
        """正向传播: F₀ → V₁ → F₁ → ... → F_goal"""
        context = {}
        
        for stage in chain.stages:
            if stage.stage_type == 'F':
                # 固定阶段: 从上游变量 + 物理公式 → 计算[min, max]
                for fc in stage.fixed:
                    fc.value_min, fc.value_max = self.compute_bounds(
                        fc.name, context)
                    context[fc.name] = (fc.value_min, fc.value_max)
                    
            elif stage.stage_type == 'V':
                # 变化阶段: 展开选择空间, 受上游固定约束限制
                for vc in stage.variables:
                    # 上游约束可能收窄选择范围
                    vc.range_min, vc.range_max = self.narrow_range(
                        vc.name, vc.range_min, vc.range_max, context)
                    context[vc.name] = vc.current  # 用户选择的值
        
        return context
    
    def reverse_propagate(self, chain: ConstraintChain, goal: dict):
        """反向传播: 从F_goal需求 → 反推每个V阶段的允许范围"""
        # 用户说: 承重≥20kg, 尺寸精度±0.5mm
        # 反推: 哪些材料+工艺组合能满足?
        
        feasible = []
        for material in self.all_materials():
            for structure in self.all_structures():
                # 正向传播一次
                chain_instance = self.instantiate(chain, material, structure)
                result = self.propagate(chain_instance)
                
                # 检查是否满足目标
                if self.meets_goal(result, goal):
                    feasible.append({
                        'material': material,
                        'structure': structure,
                        'margins': self.compute_margins(result, goal),
                    })
        
        # 按裕度排序 (裕度大 = 更安全)
        return sorted(feasible, key=lambda x: x['margins'], reverse=True)
    
    def compute_bounds(self, param_name, context):
        """用物理公式计算参数的[min, max]"""
        
        if param_name == 'springback_angle':
            # 回弹角 = f(E, t, R, 压痕深度)
            E_min, E_max = context['E_modulus']
            t_min, t_max = context['thickness']
            depth = context['crease_depth_ratio']
            R = 2 * t_min * (1 - depth)
            
            K_min = 1 - (R / (R + t_max))
            K_max = 1 - (R / (R + t_min))
            
            # 最小回弹(深压痕+薄板) 到 最大回弹(浅压痕+厚板)
            spring_min = math.radians(90) * (1 - K_max) * 0.02 * (1 - depth)
            spring_max = math.radians(90) * (1 - K_min) * 0.05 * (1 - depth)
            
            return math.degrees(spring_min), math.degrees(spring_max)
        
        elif param_name == 'stack_load':
            # 堆叠承重 = 壁面面积 × 压溃强度 × 安全系数 × 粘合修正
            area = context['wall_area']
            crush_min, crush_max = context['crush_strength']
            safety = 0.6  # 安全系数
            glue_factor = context.get('glue_strength_factor', 0.85)
            
            F_min = area * crush_min * safety * glue_factor
            F_max = area * crush_max * safety * 1.0  # 粘合完美时
            
            return F_min / 1000, F_max / 1000  # 转换为kg
        
        # ... 更多物理公式
```

### 4.3 优化器: 在变量空间中寻找最优

```python
class DesignOptimizer:
    """在固定约束下, 调整变量获得最优用户体验"""
    
    def optimize(self, goal, constraints):
        """
        goal: {承重: ≥20kg, 精度: ±0.5mm, 成本: ≤¥5}
        constraints: F₀ (物理定律) + F₁ (材料已选)
        
        调整: V₂ (结构参数) + V₃ (工艺参数)
        目标: 最大化用户体验 = w₁×承重裕度 + w₂×精度裕度 + w₃×(1/成本)
        """
        
        # 变量空间
        variables = {
            'wall_height': (30, 100),       # mm
            'ear_width': (10, 30),          # mm
            'crease_depth': (0.3, 0.9),     # 比率
            'corner_angle': (30, 60),       # 度
            'glue_tab_width': (10, 25),     # mm
        }
        
        best = None
        best_score = -inf
        
        # 网格搜索 / 贝叶斯优化 / 遗传算法
        for point in self.search_space(variables):
            # 正向传播: 计算此参数组合的结果
            chain = self.build_chain(constraints, point)
            result = self.engine.propagate(chain)
            
            # 检查是否满足硬约束
            if not self.meets_hard_constraints(result, goal):
                continue
            
            # 计算用户体验分数
            score = self.ux_score(result, goal)
            
            if score > best_score:
                best_score = score
                best = point
        
        return best, best_score
    
    def ux_score(self, result, goal):
        """用户体验 = 多目标加权"""
        load_margin = (result['stack_load_min'] - goal['load']) / goal['load']
        precision = 1 - abs(result['dimension_error_max']) / goal['precision']
        cost_eff = 1 - result['cost'] / goal['max_cost']
        aesthetics = result['fold_clarity'] * result['surface_quality']
        
        return (0.3 * load_margin + 
                0.25 * precision + 
                0.20 * cost_eff + 
                0.25 * aesthetics)
```

## 五、全链路: 2D→刀模→切压→2D盒片→粘贴→3D

### 5.1 每一步的固定/变化映射

```
步骤1: 2D图纸设计

│ 变化V₁: 尺寸L/W/H (用户自由调整)                    │
│ 变化V₁: 折耳宽度/粘口宽度/角部处理方式               │
│                                                      │
│ 输出F₁: 精确的2D展开图 (数学确定)                    │
│        每条线的坐标是固定的                           │
│        每个面积是固定的                               │
│        折痕补偿量是固定的 (= 厚度×π×角度/360)        │
└────────────────────────────────────────────────────┘
                    ↓
步骤2: 刀模制造
┌────────────────────────────────────────────────────┐
│ 固定F₁: 2D图纸 (不可改)                             │

│                                                      │

│        刀线位置精度: ±0.05mm (激光切割)              │

│        桥位位置: 固定 (已制造)                        │

                    ↓
步骤3: 刀模切割压制原材料
┌────────────────────────────────────────────────────┐
│ 固定F₂: 刀模 (不可改)                               │
│ 固定F₁: 材料特性 (E, σ, ε已确定)                    │
│ 变化V₃: 模切压力/速度/补偿垫                         │
│                                                      │
│ 过程:                                                │
│   刀线切入材料 → 切断纤维 → 切口                     │

│                                                      │

│        切口精度: ±(0.05+0.10) = ±0.15mm             │
│        压痕深度: 固定 (已压制)                        │
│        回弹角: 可计算 = f(E, t, 压痕深度)            │
└────────────────────────────────────────────────────┘
                    ↓
步骤4: 折叠粘贴成型
┌────────────────────────────────────────────────────┐
│ 固定F₃: 2D盒片 (不可改)                             │

│   沿折痕弯折 → 纤维外层拉伸/内层压缩                 │

│   涂胶 → 粘合 → 固化                                │
│   固化不足 → 开胶                                    │
│   固化过度 → 脆化                                    │
│                                                      │
│ 输出F₄: 3D包装盒成品                                 │
│        外部尺寸: ±Σ(所有步骤误差) 累积                │
```

### 5.2 误差累积公式

```
最终尺寸误差 = √(ε₁² + ε₂² + ε₃² + ε₄²)   [RSS方法]

其中:
ε₁ = 2D图纸误差 ≈ 0 (数学精确)
ε₂ = 刀模制造误差 ≈ ±0.05mm (激光切割)
ε₃ = 模切误差 ≈ ±0.15mm (机器+材料)
ε₄ = 折叠粘贴误差 ≈ ±0.2mm (卡纸) / ±1.0mm (瓦楞)

白卡纸总误差: √(0² + 0.05² + 0.15² + 0.2²) = ±0.26mm
E瓦楞总误差:  √(0² + 0.05² + 0.15² + 1.0²) = ±1.02mm
```

### 5.3 承重传播公式

```
最终承重 = 材料压溃强度 × 有效承压面积 × 结构效率 × 粘合系数 × 环境修正

其中:
材料压溃强度: F_crush (材料选择后固定)
有效承压面积: A_eff = f(L, W, H, 壁厚) (结构选择后固定)
结构效率: η = 0.4~0.8 (取决于盒型, 四角有支撑=高效率)
粘合系数: κ = 0.7~1.0 (取决于胶水+固化时间)
环境修正: μ = 1.0 - 0.015×(MC - MC_base) (含水率每升高1%承重降1.5%)

白卡纸 300gsm, 120370托盘, 300×200×50:
  F = 5.5 × 600 × 0.6 × 0.85 × 0.97 = 1636 N ≈ 167 kg (顶压)

E瓦楞, 120370托盘, 300×200×50:
  F = 30 × 600 × 0.6 × 0.85 × 0.95 = 8721 N ≈ 889 kg (顶压)
```

## 七、核心结论

### 7.1 DiePre AI推演的本质

> **我们推演的不是"做什么", 而是"怎么用固定的去调整不固定的"**

- 物理定律固定 → 我们不能改变
- 材料特性固定(选择后) → 我们可以选择哪种材料
- 3D目标固定 → 客户需求不可让步
- **我们能调整的只有变量空间V₁V₂V₃V₄中的参数**
- **推演的价值 = 在变量空间中找到最优路径, 使F₄ ≥ F_goal**

### 7.2 框架对架构的约束

```
DiePre AI的每个功能模块必须明确:
1. 它接收哪些固定约束作为输入? (不可违反)
2. 它暴露哪些变量供用户/AI调整? (自由度)
3. 它输出哪些新的固定约束给下游? (锁定范围)
4. 它的[min, max]范围如何计算? (物理公式)
```

### 7.3 这超越了Pacdora

Pacdora只做了链路的前半段(2D图纸→3D可视化), 完全不涉及:
- 材料特性约束传播
- 误差累积计算
- 承重预测
- 工艺参数影响
- 逆向目标求解

**DiePre AI的护城河 = 全链路固定/不固定约束传播**
