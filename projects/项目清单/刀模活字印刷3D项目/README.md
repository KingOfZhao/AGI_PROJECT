## 一、项目目标

将传统钢规刀模（Steel Rule Die）的制作思路，结合**活字印刷**和**华容道**的模块化理念，
用 **3D 打印** 制造可复用的模块化刀模组件，实现：

1. **CAD 二维图纸 → 活字模块拆解**: 将完整刀版展开图按公式/定律拆解为标准化小模块
2. **模块化拼接 → 完整刀模**: 像活字印刷一样，用标准模块拼接成任意形状的刀模
3. **3D 打印制造**: 用拓竹 P1S/P2S 打印模块，支持各种尺寸刀片和卡纸材料
4. **可复用刀模库**: 模块标准化后可反复使用，降低刀模成本

## 二、核心概念

### 2.1 活字印刷思想
- **Gutenberg 合金**: 铅(Pb) + 锡(Sn) + 锑(Sb) = 82:9:9，熔点低、硬度适中、收缩率小
- **模块化**: 每个字模独立铸造、可任意排列组合
- **标准化**: 字模尺寸统一（字身、字面、字脚），可互换
- **华容道**: 在有限空间内排列组合模块，解决拼接约束问题

### 2.2 IADD 钢规刀模标准
| 参数 | 规格 | 说明 |
|------|------|------|
| 刀片厚度 (Point) | 2pt=0.71mm, 3pt=1.07mm, 4pt=1.42mm | IADD/Colvin-Friedman 标准 |
| 刀片高度 | 23.8mm (0.937") 标准 | 也有 22mm, 30mm, 50mm, 60mm, 80mm |
| 刀锋角度 | 中心斜角 (CB), 长中心斜角 (LCB), 侧斜角 (SB) | 不同材料用不同角度 |
| 刀板材质 | 胶合板 (Birch Plywood) / 亚克力 / 复合材料 | 激光切割槽道 |
| 公差 | ±0.254mm (IADD) | 模切精度要求 |
| 弹性橡胶 | 开孔/闭孔泡沫 | 弹料脱模 |
| 桥接 (Bridge) | 刀片中断处保持刀板完整性 | 结构强度 |

## 三、模块化拆解原理

### 3.1 模块类型定义
| 模块类型 | 功能 | 3D打印件 |
|----------|------|----------|
| **直线段 (Straight)** | 直线切割/压痕 | 刀片夹持槽 + 底座 |
| **圆弧段 (Arc)** | 曲线切割 | 带弧度的刀片夹持槽 |
| **转角 (Corner)** | 90°/45°/任意角度转弯 | 角度连接件 |
| **T形接头 (T-Joint)** | 三向交汇 | 三通连接件 |
| **十字接头 (Cross)** | 四向交汇 | 十字连接件 |
| **端头 (End Cap)** | 刀片终止 | 封口件 |
| **桥接 (Bridge)** | 刀片中断，保持结构 | 桥接连接件 |
| **弹料座 (Ejector)** | 安装弹性橡胶 | 弹料固定座 |
| **底板块 (Base Tile)** | 拼接底板 | 方形/矩形底板模块 |

### 3.2 拆解算法
```
输入: DXF/DWG 二维图纸
  ↓
Step 1: 解析几何实体 (LINE/ARC/CIRCLE/POLYLINE)
  ↓
Step 2: 分类实体 (CUT切割线 / CREASE压痕线 / GUIDE辅助线)
  ↓
Step 3: 图论分析 — 构建线段连接图 (节点=交点, 边=线段)
  ↓
Step 4: 路径分解 — 将连续路径拆解为标准模块序列
  ↓
Step 5: 模块匹配 — 每段路径匹配最优模块 (直线/圆弧/转角...)
  ↓
Step 6: 约束检查 — 3D打印尺寸约束 (≤256mm) / 刀片规格匹配
  ↓
Step 7: 拼接方案生成 — 华容道式排列优化
  ↓
Step 8: 输出 — 模块清单 + 3D打印STL + 拼接指南
```
