> 推演时间: 2026-03-30 07:16
> 原则: 最优效果下的最低成本

## 一、固定框架（不可动摇）

| # | 固定项 | 理由 |
|---|--------|------|
| F1 | 品牌名"余香" + Slogan"赠人玫瑰手有余香" | 核心标识，一旦传播不可更改 |
| F2 | 广告收入100%进入玫瑰基金 | 信任基石，公开承诺不可变 |
| F3 | 不存储身份证明文 | 合规红线 + 隐私承诺 |
| F4 | 财务全程透明公示 | 反抹黑核心武器 |
| F5 | 以私心取余香的真诚语调 | 品牌人格，变则失真 |
| F6 | 首发上海嘉定区 | 已定 |
| F7 | 线上广告收入达1000人玫瑰金额后启动线下 | 成本控制红线 |
| F8 | 净利分配比例 20:30:20:30 | 合伙协议约定 |

## 三、行进路线（6阶段）

```
阶段0: 基础搭建（第1-2周）
  └─ 注册主体 ← 人类今天开始
  └─ 商标查询/注册
  └─ 小程序开发（核心功能）
  └─ 公众号注册
  └─ APP原型（验证流程）
  └─ 合伙协议签署

阶段1: 种子传播（第2-4周）
  └─ 小程序上线（报名+声明+广告）
  └─ 自媒体账号开通（小红书/抖音/快手）
  └─ 发布首批内容：品牌声明 + 视觉 + "为什么要做余香"
  └─ 目标：小程序注册用户突破100
  └─ 成本：服务器~50元/月，无其他支出

阶段2: 增长验证（第4-8周）
  └─ 持续内容输出（每周3-5条）
  └─ 观察各平台数据，确定主力平台
  └─ 优化广告配置，测试收益
  └─ 目标：广告收入开始产生，用户破500
  └─ 成本：服务器+内容制作工具~200元/月

阶段3: 千人里程碑（第8-12周）
  └─ 广告累计收入 ≥ 1000朵玫瑰成本
  └─ 触发线下发放准备
  └─ 锁定嘉定区发放点
  └─ 玫瑰供应商确定
  └─ APP验证工具上线
  └─ 目标：1000朵玫瑰采购+包装到位

阶段4: 首次线下发放（上海嘉定）
  └─ 现场发放：身份证+二维码验证
  └─ 拍摄记录，回传素材
  └─ 公众号公示：发放数、成本、现场情况
  └─ 线上同步放大（发放视频/故事）
  └─ 目标：首批发放成功，形成可复制模式

阶段5: 循环放大
  └─ 根据首次效果调整
  └─ 扩展发放点/城市
  └─ 申请CTID接入
  └─ 品牌合作探索
```

## 四、成本估算（最低成本方案）

| 阶段 | 支出项 | 月成本 |
|------|--------|--------|
| 0 | 主体注册+商标 | 一次性~3000元 |
| 0-1 | 服务器（轻量云） | ~50元/月 |
| 1 | 自媒体运营（纯内容，不投流） | 0元 |
| 2 | 内容制作工具 | ~100元/月 |
| 3 | 1000朵玫瑰+包装 | 预估2000-3000元（来自广告收入） |
| **线上总投入** | | **~1500元（2个月）+注册费** |
| **关键点** | 玫瑰成本由广告收入覆盖 | 0元自掏 |

**最优效果最低成本 = 零投放纯内容增长，用时间换流量。**
