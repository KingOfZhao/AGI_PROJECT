## 调用链路

```
用户提问 → Ollama路由 → GLM-5 Turbo快速分析 → GLM-5深度推理
→ GLM-4.7代码生成 → Ollama幻觉校验 → 零回避扫描 → 返回结果
```

## 已知阻塞问题 (6个)

1. **DXF解析器复杂曲线支持** (刀模) — 需要研究libdxfrw或自实现贝塞尔
2. **材料物理参数缺乏实测数据** (刀模) — 需联系厂商或试验机
3. **支付牌照/合规** (予人玫瑰) — 接入微信支付/支付宝
4. **拓竹P2S打印精度限制** (活字印刷) — ±0.2mm, 可考虑SLA
5. **本地算力限制** (模型超越) — GLM-5 Turbo补充
6. **算子完备性证明难度** (三算子) — 需数学AI辅助

## Shell 安全防护

所有推演过程中生成的shell命令经过安全检查, **禁止**以下模式:
- `dquote>` 双引号未闭合
- `quote>` 单引号未闭合
- `>` 管道/括号续行
- `heredoc>` Here-Document未结束
- `` `> `` 反引号未闭合
