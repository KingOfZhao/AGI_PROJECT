### 3. 单次OODA执行
```bash

python3 workspace/ooda_controller.py --model qwen2.5-coder:7b "实现快速排序算法"
```

### 4. Python API调用
```python
from workspace.ooda_controller import DualLoopEngine

engine = DualLoopEngine(model="qwen2.5-coder:14b")

# 单次执行
result = engine.run("实现一个Fibonacci计算器")
print(result["status"])     # "done"
print(result["path"])       # ["v1_coding", "v4_validate"]
print(result["duration_ms"])

# 批量执行
results = engine.run_batch([
    "写一个排序算法",
    "构建知识图谱",
    "生成一个新技能: PDF解析器",
])

# 查看全局态势
status = engine.status()
print(status["cop"]["tasks"])       # 任务统计
print(status["cop"]["knowledge"])   # 知识库统计
print(status["cop"]["cache"])       # 缓存命中率

engine.close()
```

## 二、核心组件

### COP 共享态势层
```python
from workspace.cop_shared_awareness import COP

cop = COP()

# 任务管理
task_id = cop.tasks.create_task("实现XX功能", source="L0")
cop.tasks.update_status(task_id, "running", assigned_to="v1")
cop.tasks.update_status(task_id, "done", result="代码输出...")
pending = cop.tasks.get_pending()
summary = cop.tasks.summary()  # {"pending": 0, "running": 1, "done": 2, "failed": 0}

# 知识存储 (三层)
nid = cop.knowledge.add("原始执行日志", layer=1, importance=0.3)       # 7天TTL
nid = cop.knowledge.add("结构化节点", layer=2, importance=0.6)         # 30天TTL
nid = cop.knowledge.add("精炼知识(已验证)", layer=3, importance=0.9)   # 永久

# 知识搜索
results = cop.knowledge.search("Fibonacci", layer=2, limit=10)

# 引用计数 (提升重要度)
cop.knowledge.reference(nid)

# 遗忘引擎 (清理过期+低重要度)
forgotten = cop.knowledge.run_forgetting()

# 路径缓存 (加速OODA Orient)
cop.cache.put("实现算法函数", ["v1_coding", "v4_validate"])
cached = cop.cache.get("实现算法函数")  # ["v1_coding", "v4_validate"]
print(cop.cache.hit_rate)               # 0.80+

# 冲突检测
conflicts = cop.conflicts.check_task_conflicts()

# 事件订阅
cop.bus.subscribe("task.done", lambda e: print(f"任务完成: {e}"))

# OODA日志
cop.log_ooda("cycle_001", "observe", "输入摘要", "输出摘要", 50, "v1_coding")
history = cop.get_ooda_history(limit=20)

# 全局态势
status = cop.full_status()

cop.close()
```

### OODA 内环控制器
```python
from workspace.cop_shared_awareness import COP
from workspace.ooda_controller import OODAController, SkillLoader

cop = COP()
skills = SkillLoader(model="qwen2.5-coder:14b")

print(skills.available())  # ['v1_coding', 'v2_orchestrate', 'v3_generate', ...]

ooda = OODAController(cop, skills)
result = ooda.run_cycle("写一个Web爬虫")

#   "results": {"v1_coding": "...", "v4_validate": "..."},
#   "duration_ms": 38000,
#   "status": "done"
# }
```

### 进化外环
```python
from workspace.ooda_controller import EvolutionLoop

evo = EvolutionLoop(cop, skills, trigger_every=5)

# 每次OODA完成后调用tick
for intent in task_list:
    ooda.run_cycle(intent)
    evo.tick()  # 每5次自动触发进化
```

## 三、意图→技能路由

| 意图关键词 | 路由路径 | 说明 |
|-----------|---------|------|
| 编码/代码/函数/算法/实现 | v1→v4 | 编码+验证 |
| 项目 | v1→v5→v4 | 编码+脚手架+验证 |
| 脚手架 | v5→v4 | 项目搭建+验证 |
| 知识/图谱/知识图谱 | v6 | 知识节点构建 |
| 生成技能 | v3→v4 | 新技能生产+验证 |
| 进化 | v7→v4 | 自主进化+验证 |
| 验证 | v4 | 纯验证 |
| (默认) | v1→v4 | 编码+验证 |

路径缓存: 相同意图第二次执行时自动命中缓存, 跳过路由计算。
