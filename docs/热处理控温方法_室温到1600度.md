## 一、总体控温策略

### 1.1 温度段划分原则

从室温到1600°C，按**相变临界点**和**控制难度**划分为6个温度段：

| 温度段 | 范围 | 关键相变/事件 | 控制难度 |
|--------|------|---------------|----------|
| **Ⅰ 低温段** | 25~200°C | 去应力退火区 | ★☆☆☆☆ 简单 |
| **Ⅱ 中低温段** | 200~500°C | 回火区(马氏体分解) | ★★☆☆☆ 较易 |
| **Ⅲ 中温段** | 500~727°C | 接近A1线(共析温度727°C) | ★★★☆☆ 中等 |
| **Ⅳ 中高温段** | 727~912°C | A1→A3相变区(α→γ) | ★★★★☆ 较难 |
| **Ⅴ 高温段** | 912~1148°C | 奥氏体区→共晶温度 | ★★★★★ 困难 |
| **Ⅵ 超高温段** | 1148~1600°C | 液相出现→纯铁熔点(1538°C) | ★★★★★ 极难 |

## 二、各温度段控温方法

---

### Ⅰ 低温段：25°C → 200°C

**典型工艺**: 去应力退火、低温回火、预热

#### 控温目标

| 精度等级 | 温差范围 | 适用场景 |
|----------|----------|----------|
| 一般 | ±20°C | 预热、干燥 |
| 标准 | ±10°C | 去应力退火 |
| 精密 | ±5°C | 精密零件预处理 |

#### 控温方法

**方法A：开关控制(On-Off)**
- 原理：温度低于设定值→全功率加热，高于设定值→断电
- 适用：±20°C精度，简单炉体
- 设备：温控仪 + 继电器 + K型热电偶
- 🔧 PID参数：不需要（纯开关）

**方法B：时间比例控制**
- 原理：在固定周期(如10s)内，按偏差比例分配加热时间
- 适用：±10°C精度
- 设备：固态继电器(SSR) + 温控仪
- 🔧 周期建议：T_cycle = 5~15s

**方法C：简单PID控制**
- 原理：比例+积分+微分三项控制
- 适用：±5°C精度
- 🔧 建议PID参数：
  ```
  Kp = 8.0    (比例增益)
  Ki = 0.05   (积分增益, 1/s)
  Kd = 2.0    (微分增益, s)
  采样周期: 500ms
  ```

#### ⚠️ 注意事项
- 升温速率建议：2~5°C/min（避免热应力）
- 低温段热惯性小，响应快，过冲风险低
- 热电偶精度在低温段较高(±1.5°C for K型)

#### 🔧 升温曲线计算

```
阶段1: 25°C → 200°C
  升温速率: 3°C/min
  升温时间: (200-25)/3 = 58.3 min ≈ 60 min
  保温时间: 按工艺要求(去应力退火通常1~2h)
```

### Ⅱ 中低温段：200°C → 500°C

**典型工艺**: 回火(低温/中温)、时效处理

#### 控温目标

| 精度等级 | 温差范围 | 适用场景 |
|----------|----------|----------|
| 标准 | ±10°C | 一般回火 |
| 精密 | ±5°C | 弹簧/模具回火 |
| 高精 | ±3°C | 精密量具回火 |

#### 回火温度与硬度关系(中碳钢, proven)

| 回火温度 | 组织 | 硬度(HRC) | 用途 |
|----------|------|-----------|------|
| 150~250°C | 回火马氏体 | 58~64 | 切削工具、量具 |
| 250~400°C | 回火托氏体 | 40~50 | 弹簧、模具 |
| 400~500°C | 回火索氏体 | 25~35 | 连杆、齿轮轴 |

#### 控温方法

**方法A：PID控制 + 分段升温**
- 原理：将200→500分为2~3段，每段独立PID
- 🔧 建议PID参数：
  ```
  200~350°C段:
    Kp = 6.0, Ki = 0.04, Kd = 3.0
    升温速率: 3~5°C/min
  
  350~500°C段:
    Kp = 5.0, Ki = 0.03, Kd = 4.0
    升温速率: 2~4°C/min
  ```

**方法B：模糊PID**
- 原理：根据偏差e和偏差变化率ec实时调整Kp/Ki/Kd
- 适用：±3°C精度
- 模糊规则表(简化)：
  ```
  当 e大且ec正 → Kp↑ Ki↓ Kd↑ (快速响应)
  当 e小且ec小 → Kp适中 Ki适中 Kd适中 (精细控制)
  当 e≈0且ec≈0 → Kp↓ Ki↑ Kd↓ (稳态保持)
  ```

**方法C：程序控温(多段式)**
- 适用：需要特定升温-保温-冷却曲线的工艺
- 设备：可编程温控仪(如Eurotherm 3216/Omron E5CC)
- 程序段示例：
  ```
  段1: 25°C → 200°C, 速率3°C/min, 保温0min
  段2: 200°C → 350°C, 速率4°C/min, 保温30min(中间保温消除温度梯度)
  段3: 350°C → 目标温度, 速率3°C/min, 保温按工艺
  ```

#### ⚠️ 注意事项
- **回火脆性警告**: 350~400°C区间为第一类回火脆性区，某些钢种应快速通过
- 保温时间一般按截面厚度计算：1~2 min/mm
- 👤 实际回火硬度需用硬度计验证

### Ⅲ 中温段：500°C → 727°C

**典型工艺**: 高温回火、球化退火、再结晶退火

#### 控温目标

| 精度等级 | 温差范围 | 适用场景 |
|----------|----------|----------|
| 标准 | ±10°C | 高温回火 |
| 精密 | ±5°C | 球化退火 |
| 高精 | ±3°C | 接近A1线操作 |

#### ⚠️ 关键：接近A1线(727°C)控制

727°C是共析转变温度(A1线)，在此温度附近：
- **低于727°C**: 组织为铁素体+渗碳体(珠光体)
- **高于727°C**: 开始向奥氏体转变
- **球化退火**: 需在700~720°C(A1以下)长时间保温，使片状渗碳体球化

#### 控温方法

**方法A：精密PID + 慢速接近**
- 策略：距目标30°C以内时，降低升温速率到0.5~1°C/min
- 🔧 建议PID参数：
  ```
  500~690°C段:
    Kp = 4.5, Ki = 0.03, Kd = 5.0
    升温速率: 2~3°C/min
  
  690~727°C段(精密接近):
    Kp = 3.0, Ki = 0.02, Kd = 8.0
    升温速率: 0.5~1°C/min
    采样周期: 200ms
  ```

**方法B：双PID串级控制**
- 外环：工件温度PID → 输出目标炉温
- 内环：炉温PID → 控制加热功率
- 优势：补偿工件-炉温滞后，减小工件实际温差
- 设备：需2个热电偶(炉内+工件接触)

**方法C：前馈+PID复合控制**
- 原理：根据热时间常数预测温度变化，提前调整功率
- 🔧 热时间常数估算(proven公式)：
  ```
  τ = ρ·c_p·V / (h·A)
  
  参数:
    ρ: 材料密度(钢≈7800 kg/m³)
    c_p: 比热容(钢≈500 J/(kg·K))
    V: 工件体积(m³)
    h: 对流换热系数(自然对流≈5~25 W/(m²·K), 强制≈25~250)
    A: 换热面积(m²)
  
  示例(∅50×100mm圆柱):
    V = π·0.025²·0.1 = 1.96×10⁻⁴ m³
    A = 2π·0.025²+π·0.05·0.1 = 1.96×10⁻² m²
    τ = 7800×500×1.96e-4 / (50×1.96e-2) ≈ 780s ≈ 13min
  ```

#### ⚠️ 注意事项
- 距A1线(727°C)±10°C范围内操作需特别谨慎
- 球化退火要求温度波动<±5°C，否则影响球化效果
- 此温度段辐射换热开始显著，需考虑炉内温度均匀性
- 👤 建议增加工件接触式热电偶测温

### Ⅳ 中高温段：727°C → 912°C

**典型工艺**: 退火、正火、淬火加热、渗碳

#### 控温目标

| 精度等级 | 温差范围 | 适用场景 |
|----------|----------|----------|
| 标准 | ±10°C | 一般退火/正火 |
| 精密 | ±5°C | 淬火加热 |
| 高精 | ±3°C | 精密淬火/等温淬火 |

#### ⚠️ 关键相变区

此段跨越A1(727°C)到A3(912°C for 纯铁)，是最重要的热处理区间：

| 碳含量 | A1温度 | A3/Acm温度 | 推荐加热温度 | 工艺 |
|--------|--------|------------|-------------|------|
| 0.20%C | 727°C | ~855°C | 870~900°C | 正火 |
| 0.45%C | 727°C | ~810°C | 830~860°C | 淬火 |
| 0.60%C | 727°C | ~780°C | 790~820°C | 淬火 |
| 0.77%C | 727°C | 727°C | 760~790°C | 淬火 |
| 1.00%C | 727°C | Acm≈810°C | 770~800°C(亚温淬火) | 淬火 |

#### 控温方法

**方法A：分段PID + 相变补偿**
- 关键：跨越A1线时材料吸热(潜热)导致温度突然停滞
- 策略：检测到温度上升停滞时，增大加热功率补偿相变潜热
- 🔧 建议PID参数：
  ```
  727~800°C段(相变区):
    Kp = 5.0, Ki = 0.04, Kd = 6.0
    相变补偿: 检测dT/dt < 0.2°C/min时, 功率+20%
    升温速率: 1~2°C/min
  
  800~912°C段(奥氏体区):
    Kp = 4.0, Ki = 0.03, Kd = 5.0
    升温速率: 2~3°C/min
  ```

**方法B：MPC模型预测控制**
- 原理：建立炉温-工件温度传递函数模型，预测未来N步温度
- 优势：可提前补偿相变潜热、热惯性滞后
- 🔧 简化模型：
  ```
  T_work(k+1) = T_work(k) + Δt/τ · [T_furnace(k) - T_work(k)] - Q_phase/mc_p
  
  其中:
    Δt: 控制周期
    τ: 热时间常数
    Q_phase: 相变潜热(α→γ约 16 kJ/kg)
    m: 工件质量
    c_p: 比热容
  ```
- 预测时域: N=10~20步(每步1s)
- 控制时域: M=3~5步

**方法C：多区独立控制**
- 将炉膛分为上/中/下(或前/中/后)多个加热区
- 每区独立PID + 主控协调
- 适用：大型箱式炉/井式炉
- 设备：多组加热元件 + 多点测温

#### 🔧 碳扩散系数计算(proven公式 — Arrhenius方程)

在此温度段进行渗碳时，碳在奥氏体中的扩散系数：

```
D = D₀ · exp(-Q/(R·T))

参数:
  D₀ = 2.0×10⁻⁵ m²/s (频率因子)
  Q = 142,000 J/mol (活化能)
  R = 8.314 J/(mol·K) (气体常数)
  T = 绝对温度(K)

示例:
  T = 900°C = 1173K
  D = 2e-5 × exp(-142000/(8.314×1173))
  D = 2e-5 × exp(-14.55)
  D ≈ 2e-5 × 4.82e-7
  D ≈ 9.64×10⁻¹² m²/s

  T = 850°C = 1123K
  D ≈ 5.12×10⁻¹² m²/s (降50°C扩散系数几乎减半)
```

**结论**: 温度每降低50°C，碳扩散系数下降约40~50%。控温精度直接影响渗碳深度和均匀性。

#### ⚠️ 注意事项
- 淬火加热温度过高→晶粒粗大→力学性能下降
- 淬火加热温度过低→未完全奥氏体化→硬度不足
- 升温通过A1线时有相变潜热吸收，需注意补偿
- 保温时间：一般按工件有效厚度 1.5~2 min/mm
- 👤 淬火冷却介质(水/油/盐浴)选择需根据钢种和尺寸

### Ⅴ 高温段：912°C → 1148°C

**典型工艺**: 渗碳、固溶处理、高温扩散退火、特殊钢热处理

#### 控温目标

| 精度等级 | 温差范围 | 适用场景 |
|----------|----------|----------|
| 标准 | ±10°C | 一般渗碳 |
| 精密 | ±5°C | 精密渗碳/固溶 |
| 高精 | ±3°C | 航空材料热处理 |

#### ⚠️ 关键温度点

| 温度 | 事件 | 影响 |
|------|------|------|
| 912°C | A3(纯铁γ→δ起始) | — |
| 930°C | 渗碳常用温度上限 | 超过→晶粒粗大 |
| 1000°C | 高温渗碳/扩散退火 | 需真空或保护气氛 |
| 1050°C | 不锈钢固溶温度 | — |
| 1100°C | 高速钢淬火温度 | — |
| 1148°C | 共晶温度(E点) | 超过→铸铁出现液相 |

#### 控温方法

**方法A：自适应PID + 温度前馈**
- 高温段热辐射主导(Stefan-Boltzmann: Q ∝ T⁴)，换热特性非线性
- 需自适应调整PID参数以匹配非线性
- 🔧 自适应规则：
  ```
  912~1000°C:
    Kp = 3.5, Ki = 0.025, Kd = 6.0
    采样周期: 200ms
  
  1000~1148°C:
    Kp = 3.0, Ki = 0.02, Kd = 8.0
    采样周期: 100ms
    前馈补偿: ΔP = k_ff · (T_target - T_current) / T_current
    k_ff = 0.15 (前馈系数)
  ```

**方法B：MPC + 辐射换热模型**
- 🔧 辐射换热公式(proven — Stefan-Boltzmann)：
  ```
  q_rad = ε·σ·(T_heater⁴ - T_work⁴)
  
  参数:
    ε: 综合发射率(钢氧化表面≈0.7~0.9)
    σ = 5.67×10⁻⁸ W/(m²·K⁴) (Stefan-Boltzmann常数)
  
  示例(1000°C加热):
    T_heater = 1050°C = 1323K
    T_work = 1000°C = 1273K
    q_rad = 0.8 × 5.67e-8 × (1323⁴ - 1273⁴)
    q_rad ≈ 0.8 × 5.67e-8 × (3.06e12 - 2.62e12)
    q_rad ≈ 0.8 × 5.67e-8 × 4.4e11
    q_rad ≈ 19,958 W/m² ≈ 20 kW/m²
  ```

**方法C：串级控制 + 均热板/辐射屏蔽**
- 外环：工件温度(接触式热电偶)
- 内环：炉膛温度(悬挂式热电偶)
- 均热板：在工件与加热元件之间放置SiC或金属均热板，均匀化辐射
- 适用：±3°C高精度要求

#### ⚠️ 注意事项
- **氧化脱碳**: >900°C空气中钢表面严重氧化，需保护气氛(N₂/Ar/真空/渗碳气氛)
- **晶粒长大**: >950°C奥氏体晶粒快速粗化，保温时间应严格控制
- **炉衬寿命**: >1000°C对炉衬耐火材料要求高
- 热电偶选型：K型上限约1100°C，超过需用S型(Pt-PtRh)或B型
- 👤 需定期校准热电偶(高温偏移显著)

### Ⅵ 超高温段：1148°C → 1600°C

**典型工艺**: 高速钢淬火(1200~1280°C)、陶瓷烧结、特种合金热处理、冶金熔炼

#### 控温目标

| 精度等级 | 温差范围 | 适用场景 |
|----------|----------|----------|
| 标准 | ±15°C | 一般高温处理 |
| 精密 | ±10°C | 高速钢淬火 |
| 高精 | ±5°C | 特种材料/科研 |

#### ⚠️ 极端条件

| 温度 | 事件/限制 | 应对 |
|------|-----------|------|
| 1148°C | 共晶温度，高碳钢出现液相 | 低碳钢才可超过 |
| 1200~1280°C | 高速钢淬火温度 | 需盐浴或真空炉 |
| 1394°C | γ→δ转变(纯铁) | 仅极低碳材料 |
| 1495°C | 包晶反应温度 | 冶金范畴 |
| 1538°C | 纯铁熔点 | 必须真空/惰性气氛 |
| 1600°C | 超出常规热处理范围 | 特种炉(钼丝炉/硅碳棒炉) |

#### 控温方法

**方法A：高温PID + 功率限幅 + 安全联锁**
- 🔧 建议PID参数：
  ```
  1148~1300°C:
    Kp = 2.5, Ki = 0.015, Kd = 10.0
    功率限幅: 最大85%额定功率(防过冲)
    升温速率: 1~2°C/min
    采样周期: 100ms
  
  1300~1600°C:
    Kp = 2.0, Ki = 0.01, Kd = 12.0
    功率限幅: 最大90%额定功率
    升温速率: 0.5~1°C/min(接近目标时)
    安全联锁: T>T_target+20°C → 立即断电
  ```

**方法B：双色比色测温 + PID**
- >1200°C后，接触式热电偶寿命短且易漂移
- 采用双色红外比色测温仪(非接触式)
- 精度：±(0.5%·T+1)°C
- 与接触式热电偶交叉校准

**方法C：真空炉控温**
- 真空环境下无对流换热，仅辐射换热
- 控温响应更快(无气体热容缓冲)
- 需增大微分项(Kd)防止振荡
- 🔧 真空炉PID调整：
  ```
  Kp = 2.0 (降低，因响应快)
  Ki = 0.01 (降低，防积分饱和)
  Kd = 15.0 (增大，抑制振荡)
  ```

**方法D：盐浴炉控温(1200~1300°C)**
- 介质：BaCl₂盐浴(工作范围1100~1350°C)
- 优势：温度极其均匀(盐浴内温差<±3°C)
- 加热方式：电极加热(浸没式电极)
- 控温简单：盐浴热容大，自然均温

#### 加热元件选择

| 温度范围 | 加热元件 | 最高使用温度 |
|----------|----------|-------------|
| ~1100°C | 镍铬合金丝(Cr20Ni80) | 1100°C |
| ~1300°C | 铁铬铝合金丝(0Cr27Al7Mo2) | 1300°C |
| ~1400°C | 硅碳棒(SiC) | 1400°C |
| ~1600°C | 硅钼棒(MoSi₂) | 1700°C |
| ~1800°C | 钨丝/钼丝(真空) | 2000°C(真空) |

#### 热电偶选择

| 类型 | 测温范围 | 精度 | 适用 |
|------|----------|------|------|
| K型(NiCr-NiSi) | -200~1100°C | ±1.5°C或±0.4% | 中低温段 |
| N型(NiCrSi-NiSi) | -200~1200°C | ±1.5°C或±0.4% | 替代K型 |
| S型(Pt-PtRh10) | 0~1600°C | ±1.0°C或±0.25% | 高温段 |
| B型(PtRh6-PtRh30) | 600~1700°C | ±1.5°C或±0.25% | 超高温 |
| 钨铼热电偶 | 0~2300°C | ±1%T | 真空超高温 |

#### ⚠️ 注意事项
- **安全第一**: >1200°C操作需完善安全联锁(超温断电、水冷系统、气氛监测)
- **热电偶保护**: S/B型热电偶需刚玉(Al₂O₃)保护管
- **炉体散热**: 需计算炉壁散热量，确保功率余量>30%
- **气氛控制**: 必须使用保护气氛或真空(空气中钢1200°C氧化速率极快)
- 👤 超高温操作需专业人员+完善安全措施

### 5.1 安全联锁

| 联锁项 | 触发条件 | 动作 |
|--------|----------|------|
| 超温保护 | T > T_set + 30°C | 立即断电 + 报警 |
| 热电偶断偶 | 信号开路 | 立即断电 + 报警 |
| 水冷故障 | 水流<最小值 | 降功率→断电 |
| 气氛异常 | O₂含量>限值 | 增大保护气流量 + 报警 |
| 炉门开启 | 门限位开关触发 | 断电(高温段) |

### 5.2 热电偶校准周期

| 温度范围 | 热电偶类型 | 建议校准周期 |
|----------|-----------|-------------|
| <500°C | K型 | 12个月 |
| 500~1100°C | K型/N型 | 6个月 |
| 1100~1600°C | S型/B型 | 3个月 |

### 5.3 能耗估算

🔧 加热功率估算公式：
```
P = m·c_p·ΔT/Δt + P_loss

参数:
  m: 工件质量(kg)
  c_p: 比热容(钢≈500 J/(kg·K))
  ΔT/Δt: 升温速率(°C/s)
  P_loss: 散热损失(与炉体保温有关)

示例(10kg工件, 3°C/min升温):
  P_heating = 10 × 500 × (3/60) = 250 W
  P_loss ≈ 2~10 kW (取决于炉温和保温)
  P_total ≈ 3~11 kW (实际功率以散热为主)
```
