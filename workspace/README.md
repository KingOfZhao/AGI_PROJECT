# workspace/ — 工作区

系统运行时生成的技能模块、输出文件和临时数据。

## 子目录

| 目录/文件 | 内容 |
|-----------|------|
| `skills/` | 系统自动生成和手动注入的技能模块（.py + .meta.json） |
| `outputs/` | 工具执行产生的输出文件（DXF/STEP/脚本等） |
| `logs/` | 运行日志 |
| `test_app/` | 测试应用 |

## skills/ 核心技能

| 技能 | 功能 |
|------|------|
| `zhipu_growth.py` | 智谱自成长引擎（hypothesis→proven验证循环） |
| `zhipu_ai_caller.py` | 智谱API调用封装 |
| `bodhi_path.py` | 菩提路径（经典智慧推理引擎） |
| `software_engineer.py` | 软件工程能力模块 |
| `codebase_analyzer.py` | 代码库分析器 |
| `math_formula_engine.py` | 数学公式编码引擎 |
| `cad_drawing_generator.py` | CAD图纸生成器 |
| `cad_file_recognizer.py` | CAD文件识别器 |
| `flutter_generator.py` | Flutter代码生成器 |
| `web_researcher.py` | 网络研究工具 |
| `shell_executor.py` | Shell命令执行器 |
| `tool_forge.py` | 工具锻造（动态生成新工具） |
| `surpass_engine.py` | 超越引擎（能力突破） |
| `capability_bootstrap.py` | 能力自举引导 |
