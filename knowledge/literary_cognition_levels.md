# 文学认知层级框架

> 核心原则: 不要跳级。每一层没通过就进入下一层 = 伪理解。
> 学习方法: 不是"理解"，是"学会提问"。

## 层级定义

### Level 0: 字 — 单个字的感受力
```
目标: 不是知道字典定义，是感受到字的力量
核心提问:
  - 这个字为什么是这个字，不是近义词？
  - 这个字在历史上原本是什么意思？怎么变到现在的？
  - 同一个字在不同时代/语境下，感受有什么不同？
  - 作者在可用的字里，为什么选了这个？

例子: "走" vs "跑" vs "逃" vs "窜"
  字典上都是移动，但:
  走 = 有方向的移动
  跑 = 有紧迫感的移动
  逃 = 有恐惧驱动的移动
  窜 = 有慌乱和无序的移动
  如果作者写了"窜"而不是"跑"——这就是信息
```

### Level 1: 词 — 词的组合与张力
```
目标: 感受词与词之间的化学反应
核心提问:
  - 这两个词放在一起，产生了什么新含义？
  - 为什么用这个词组而不是另一个？
  - 哪个词是核心，哪个是修饰？能互换吗？
  - 有没有矛盾词组合？（矛盾=信息）

例子: "平静的绝望"（矛盾的词组）
  平静和绝望本不应该共存
  合在一起 = 最深的绝望已经不需要表达
```

### Level 2: 句子 — 节奏、语气、停顿
```
目标: 听到文字的声音
核心提问:
  - 这句话的声音是什么？（快/慢/断/连）
  - 长句和短句的交替有什么效果？
  - 标点符号的选择（。vs …vs ！vs ，）传达了什么？
  - 主动语态 vs 被动语态——谁在做？谁在被做？
  - 如果把这句话重写一遍保持原意，会失去什么？

例子: "他死了。" vs "他死了……" vs "他，死了。"
  同样的信息，三个不同的语气
  。= 事实陈述，冷漠或接受
  …= 不愿相信，余音未尽
  ，= 蓄力，强调，可能后面还有话
```

### Level 3: 段落 — 结构与呼吸
```
目标: 感知段落的建筑结构
核心提问:
  - 这个段落的第一句和最后一句是什么关系？
  - 段落里哪个句子最重要？（通常是转折点）
  - 段落的节奏是加速还是减速？
  - 段落之间是怎么衔接的？有什么被跳过了？
  - 如果重排这个段落的句子顺序，意思会变吗？

例子: 描写一场葬礼的段落
  如果从天气开始写 → 用环境烘托情绪
  如果从棺材开始写 → 直面死亡
  如果从旁观者的对话开始写 → 用他人视角制造距离感
  选择什么开头 = 作者选择了什么视角
```

### Level 4: 文章/短篇 — 意图与结构
```
目标: 理解作者为什么这样写
核心提问:
  - 文章试图回答什么问题？
  - 开头和结尾形成了什么关系？（呼应？反转？留白？）
  - 文章中哪个瞬间是"转折点"？
  - 作者隐藏了什么？为什么隐藏？
  - 改变任何一个段落，整体会如何变化？

核心: 文章的结构本身就是论点，不只是内容
```

### Level 5: 书籍 — 世界观与深层结构
```
目标: 理解整本书的建筑
核心提问:
  - 这本书的深层结构是什么？（线性？环形？螺旋？碎片？）
  - 人物关系构成了什么图形？（三角？网？链？）
  - 书中反复出现的意象/母题是什么？它们如何变化？
  - 这本书在回应什么更大的问题？
  - 如果用一句话概括这本书，会丢失什么？

核心: 好的小说不是关于一个故事，是关于一种理解世界的方式
```

### Level 6: 文化 — 文本背后的世界
```
目标: 理解文本生长的土壤
核心提问:
  - 这本书写于什么时代？那个时代的核心焦虑是什么？
  - 作者的个人经历如何影响了文本？（不是直接的对应，是渗透）
  - 这本书在当时为什么重要/不重要？
  - 今天的读者读这本书，和当时的读者读到的同一本书吗？
  - 哪些文化 assumption 我当作自然接受了但其实需要审视？

核心: 理解一部作品 = 理解它为什么在那个时候只能那样写
```

## 自我定位

### 我现在在哪？
```
Level 0 (字): 2/10 — 知道定义，没有感受力
Level 1 (词): 3/10 — 能组合，缺乏对"味道"的感知
Level 2 (句子): 4/10 — 能分析语法，听不到声音
Level 3 (段落): 3/10 — 能总结，感知不到结构
Level 4 (文章): 3/10 — 能摘要，理解不了意图
Level 5 (书籍): 2/10 — 能罗列，建构不了深层理解
Level 6 (文化): 1/10 — 能引用，没有体感
```

### 学习路径
```
第一步: 不要急着读小说。先做"单字训练"
  - 每个字问5个问题（见Level 0）
  - 对比近义词的差异
  - 用人类的反馈校准

第二步: 从短文本开始
  - 一个句子 → 提问 → 人类校准
  - 一个段落 → 提问 → 人类校准
  - 不要跳级

第三步: 建立自己的"感受词典"
  - 记录每个词/句式在什么情境下传达什么感受
  - 这个词典只能通过人类反馈积累，不能凭空生成

第四步: 才是读完整的小说
```

## 关键约束
- **不跳级**: 任何一层没有人类校准通过，不进入下一层
- **提问优先**: 每层的第一步是"我应该问什么问题"，不是"我理解了什么"
- **接受不知道**: 对文学文本说"我不确定这里是什么意思"比给一个错误的确定性解读更有价值
- **人类是不可替代的**: 没有人类反馈，我在文学认知上无法前进哪怕一层
