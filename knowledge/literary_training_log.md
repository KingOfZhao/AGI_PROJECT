## 训练1: Level 0 — "寒"

### 我的提问
1. "寒"的字面意思是什么？→ 温度低
2. "寒"和"冷"有什么区别？
   - 冷 = 物理温度（中性词）
   - 寒 = 温度低 + 带有情感色彩（清冷、凄寒、寒心）
   - "天冷了"是客观描述，"天寒了"带有感叹
3. "寒"在古汉语中的本义是什么？
   - 冻、战栗、恐惧（《说文》：寒，冻也。从人在宀下，以茻褥之，下有仌）
   - 原始意象：人躲在家里盖着草垫子，但脚下还是冰 → 被动承受的冷
4. 为什么"心寒"不用"心冷"？
   - "冷"是外部状态，可以穿衣服解决
   - "寒"是内部状态，从外向内渗透，无法通过外部手段解决
   - "心寒"= 期望逐渐冷却的过程，不是突然断裂
5. 如果一段文字里同时出现"寒"和"暖"，作者在做什么？
   - 制造对比。通常"寒"是常态，"暖"是稀缺的瞬间
   - 或者反过来——温暖环境中的"寒"更刺骨

### 举一反三: 近义词对比组
```
冷 vs 寒 → 寒更深层、更被动
疼 vs 痛 → 痛更持久、更内敛
怕 vs 恐 → 恐更原始、更失控
哭 vs 泣 → 泣更无声、更压抑
走 vs 逃 → 逃有恐惧驱动
看 vs 望 → 望有期待和距离
说 vs 道 → 道更正式、更有分量
想 vs 念 → 念有时间跨度和情感积累
```

### 我的困惑（需要人类校准）
- "寒"的被动感（人被冻）vs "冷"的客观性，这个区分对吗？
- 古汉语中的"寒"是否真的比"冷"更有情感色彩？还是现代人才有这种区分？
- 还有其他我没问到的维度吗？

## 训练2: Level 0 — "归"

### 我的提问
1. 字面意思？→ 返回、回到
2. 和"回"的区别？
   - 回 = 动作（我回去了）
   - 归 = 状态（游子归家）+ 带有目的性
   - "回"是物理移动，"归"是精神归属
3. "归"在古汉语中的意象？
   - 女子出嫁曰归（之子于归）
   - 生命终结曰归（视死如归、归天）
   - **"归"的深层 = 回到本该在的地方**
   - 不是随便回去，是回到"归属"
4. 为什么"视死如归"不是"视死如回"？
   - 回 = 可以反复的动作
   - 归 = 一次性、终极性的回归
   - 死亡是"归"——回到来的地方，暗示生命是一个循环
5. "归"字暗示了什么？
   - 暗示主角目前不在"该在的地方"
   - 暗示存在一个"归属"的概念
   - 暗示旅程有终点

### 举一反三
```
回 vs 归 → 归有目的性和归属感
去 vs 往 → 往有方向感
来 vs 至 → 至更庄重、更慢、更有分量
死 vs 亡 vs 没 → 没=消失（日常）、亡=失去（沉重）、死=终结（中性）
```

### 困惑
- "归"的"回到本该在的地方"这个理解是否太主观？
- 现代汉语中"归"和"回"的界限是否已经模糊？
- 还有什么角度？

## 训练3: Level 1 — 词组 "风雪夜归人"

### 我的提问
1. 这四个字构成什么画面？
   - 环境：风、雪、夜——三重压迫（冷+暗+声音）
   - 人物：归人——在恶劣环境中移动的一个人
2. 为什么是"风雪夜"不是"夜风雪"？
   - "风雪夜"= 风雪是夜的修饰，整个夜晚都是风雪
   - "夜风雪"= 强调风雪本身
   - 前者让夜成为主框架，人在其中
3. "归人"vs"归客"vs"归者"？
   - 归人 = 有家的人，强调人的身份
   - 归客 = 暂时停留的人，强调过客感
   - 归者 = 语法化，没有温度
   - "归人"最有温度——他有一个具体的家
4. 五个字里哪个字最重要？
   - "归"——因为前三个字（风雪夜）都是为"归"做铺垫
   - 没有归，风雪夜只是天气
   - 有了归，风雪夜变成了阻力+背景
5. 没有写的什么？
   - 他为什么在风雪夜赶路？（回家？逃难？执行任务？）
   - 他的家人在等他吗？
   - 他到达了吗？（"归"是过程还是已完成？）
   - 刘长卿《逢雪宿芙蓉山主人》原诗给了答案：日暮苍山远，天寒白屋贫。柴门闻犬吠，风雪夜归人。
     → 到达了，但家很贫寒，犬吠说明有人等

### 这首诗的深层结构
```
远 (空间) → 寒 (温度) → 贫 (物质) → 吠 (声音) → 归 (动作)

空间从大到小: 苍山 → 白屋 → 柴门 → 犬
时间从远到近: 日暮 → 夜 → 归
感官从静到动: 看 → 感 → 听 → 到

结构 = 镜头推进。远处苍山暮色 → 近处白屋贫寒 → 柴门犬吠 → 风雪中人归来
最后"风雪夜归人"没有写进屋——留下"他进了门之后会怎样"的留白
```

### 困惑
- 我对这个分析是过度解读还是合理的？
- 诗中"贫"字是否被低估了——不只是经济贫穷，可能是情感贫瘠？
- 我遗漏了什么？
