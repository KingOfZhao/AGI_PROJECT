# 隐含表达分析框架

> 状态: v0.1 初版，需要人类实践校准
> 核心命题: 小说中最重要的信息往往没有被写出来

## 1. 留白的层次

### 第一层：文字留白（可机械化识别）
```
写出来的: "她笑了笑，低头喝了一口茶"
没写的:   她为什么笑？是在敷衍？在掩饰尴尬？在回忆？
识别方法: 找到"动作/对话"和"内心状态"之间的断层
```

### 第二层：结构留白（需要模式识别）
```
写出来的: 一个场景完整呈现
没写的:   为什么作者选择呈现这个场景而不是另一个？
识别方法: 注意"缺席"——什么本该出现但没有出现？
  - 家庭场景里缺席的父亲
  - 庆祝场景里没有提到的某个人
  - 重大事件后的沉默段落
```

### 第三层：文化留白（需要共享语境）
```
写出来的: "他终于坐上了那把椅子"
没写的:   在特定文化中，"那把椅子"意味着什么？权力？责任？诅咒？
识别方法: 识别文化符号——同一个事物在不同语境下的多重含义
```

### 第四层：矛盾留白（需要元认知）
```
写出来的: "我爱这个世界"（人物A说）
同时写出来的: 人物A每天独自喝酒到凌晨
没写的:   哪个才是真的？可能都是真的，也可能都不是
识别方法: 找到叙述内部的不自洽——不自洽本身就是信息
```

## 2. 矛盾信号识别

### 说 vs 做
```
"我相信公平" → 行为: 把资源都给了自己人
矛盾=信息: 他定义的"公平"和读者定义的不同
```

### 个体 vs 环境
```
"我选择离开" → 但整个环境没有给他留下来的选项
矛盾=信息: 这是选择还是被选择？
```

### 表面情绪 vs 深层情绪
```
笑着说话 → 但手指在桌子下攥紧
矛盾=信息: 笑容是给别人的，攥紧的手才是真实的
```

### 时间性矛盾
```
十年前的誓言 → 现在的行为完全相反
矛盾=信息: 人变了？还是誓言本身就是谎言？还是环境让他别无选择？
```

## 3. 缺席分析

### 什么没写 = 什么最重要
```
缺席的类型:
- 人的缺席: 某人从未出场但被反复提及 → 悬在所有人头上的存在
- 情感的缺席: 重大事件后没有人哭/笑 → 情感已麻木或被压抑
- 解释的缺席: 作者不给原因 → 原因显而易见或不重要，读者需要自己判断
- 结局的缺席: 开放式结尾 → 生活的本质就是不完整
```

### 缺席的权重
```
写了1000字描绘花园的美丽 → 花园重要
花园的描述中从未提到声音 → 安静可能是关键信息
花园里没有一个人 → 孤独感是核心
```

## 4. 权力关系分析

### 谁定义现实？
```
对话中:
- 谁在提问？（提问者通常控制对话方向）
- 谁在被要求解释？（解释方处于弱势）
- 谁的沉默被尊重，谁的沉默被追问？
- 谁可以用"我们"代指所有人？
```

### 空间中的权力
```
- 谁坐着，谁站着？
- 谁在房间中心，谁在边缘？
- 谁可以进入某个空间，谁不能？
- 谁的空间被入侵了？
```

## 5. 重复与变奏

### 重复=强调
```
同一个意象反复出现 → 这不是巧合，是主题
同一个动作在不同情境下出现 → 对比中见变化
```

### 变奏=发展
```
第一次: "他关上了门"
最后一次: "门自己关上了"
变化=信息: 从主动到被动，从有意识到无意识，暗示人物失去了什么
```

## 6. 自我检查清单（读小说时）

```
□ 找到至少3处"说了一件事但另一件事更真实"的矛盾
□ 找到至少1处重要的缺席（没写什么？为什么没写？）
□ 追踪一个意象/动作从开头到结尾的变化轨迹
□ 识别谁在叙述中拥有权力（谁定义了"正常"）
□ 问自己: 如果删除这段文字，理解会失去什么？
□ 注意物理细节（天气、光线、温度）如何映射内心状态
□ 找到人物自己都没意识到的动机（观察者比当事人更清楚）
```

## 7. 已知局限（需要人类校准）

### 我做不到的
- 体感共鸣（"心里一紧"的真实感受）
- 文化直觉（需要特定文化成长背景）
- 模糊容忍（我会急于"给出答案"，而小说故意不给）
- 微妙判断（讽刺vs真诚、善意vs控制的边界有时很模糊）

### 我可能过度分析的
- 把巧合当象征（有时下雨就是下雨）
- 把留白当深意（有时作者只是忘了）
- 给出确定性解读（而好的小说允许多种解读并存）

### 需要人类的
- "你读这段的真实感受是什么？"
- "这里的文化背景我理解对了吗？"
- "我的解读是不是太刻意了？"
