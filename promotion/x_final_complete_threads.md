Four-Way Collision Engine (how new Skills are born):

```python
def collide(skill_a, skill_b):
    positive = overlap(known_a, known_b)       # Shared insights
    negative = conflict(known_a, known_b)       # Contradictions
    lateral  = bridge(known_a, unknown_b)        # Cross-domain bridges
    holistic = merge(unknown_a, unknown_b)       # Emergent insights
    
    if confidence >= 95%: return new_skill()
```

Not merging. Chemical reaction. A×B → C where C is something neither A nor B could produce alone.
