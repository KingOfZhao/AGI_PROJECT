## 识别方法论推演

### 问题本质

成品盒照片 → 展开刀版图 = 3D→2D的逆向工程

| 难度 | 信息类型 | 可见性 |
|------|----------|--------|
| ★☆☆ | 外轮廓尺寸 | 可测量 |
| ★★☆ | 面板数量和排列 | 可推断 |
| ★★★ | 折痕位置(外部) | 部分可见 |
| ★★★★ | 锁扣/勾脚/锁口 | 不可见（被折叠隐藏） |
| ★★★★★ | 防撞翼片形状 | 不可见（在盒子内部） |
| ★★★★ | 压线深度/槽宽 | 不可见 |

### 推荐识别路径

#### Path A：从成品盒照片（当前图像）

```
成品盒照片
    ↓ Stage 1: 增强折痕
    ↓   - CLAHE + TopHat/BlackHat 多尺度
    ↓   - 方向性形态学滤波
    ↓ Stage 2: 面板分割
    ↓   - 检测可见面板边界
    ↓   - 利用已知尺寸约束
    ↓ Stage 3: 折痕提取
    ↓   - HoughLinesP 提取直线
    ↓   - 按角度/位置聚类
    ↓ Stage 4: 面板拓扑推断
    ↓   - 根据面板相邻关系推断展开布局
    ↓   - 已知参数约束（W/L/H/纸厚）
    ↓ Stage 5: 对称性补足
    ↓   - 利用盒子对称性填充不可见部分
    ↓   - 标记物理遮挡导致的不对称
    ↓ Stage 6: 模块推断（不可见部分）
    ↓   - 锁扣/防撞翼片：用行业公式 + 已知参数生成
    ↓   - 不能从图像验证，只能理论推导
    ↓
输出：参数化展开图（理论为主，图像验证有限）
```

**局限性**：只能验证外轮廓和可见折痕。内部结构完全靠理论。

#### Path B：从展开刀版图照片（最佳路径）

```
展开刀版图照片（平铺拍摄）
    ↓ Stage 1: 透视校正
    ↓   - 检测四角 → 仿射/透视变换
    ↓ Stage 2: 边缘提取
    ↓   - Canny + 形态学
    ↓ Stage 3: 结构识别
    ↓   - 外轮廓 → CUT层
    ↓   - 内部直线（非边缘）→ SCORE层
    ↓   - 短线段组合 → HOOK/SLOT/TONGUE层
    ↓   - 45°斜线组 → FLAP层
    ↓   - 高曲率区域 → 弧度
    ↓ Stage 4: 像素→mm换算
    ↓   - 用已知W/L/H约束换算比例
    ↓   - 或用网格校准
    ↓ Stage 5: 对称性验证
    ↓   - 左右对称：底板区域关于中心轴
    ↓   - 物理遮挡标记
    ↓ Stage 6: 精确校正
    ↓   - 用理论参数修正像素坐标
    ↓   - 对称补足缺失部分
    ↓
输出：图像验证的精确展开图
```

**优势**：所有元素都可见，可以像素级验证。

#### Path C：混合路径（推荐）

```
已知参数（W/L/H/t/K因子）
    ↓ 生成理论展开图（数学精确）
    ↓
展开刀版图照片（如有）
    ↓ 图像验证 + 对称补足
    ↓
成品盒照片（如有）
    ↓ 折痕验证（仅外部）
    ↓
输出：理论为主 + 图像辅助验证的精确展开图
```

## 技术细节：识别各模块的方法

### 1. 锁扣系统识别（HOOK/SLOT/TONGUE）

**在展开图上的特征**：
- 锁口(SLOT)：短的平行线对（U形槽口），通常5-15mm宽
- 锁舌(TONGUE)：从面板边缘延伸的矩形片，8-15mm宽×10-20mm长
- 勾脚(HOOK)：锁舌端部的回钩，2-5mm深

**识别方法**：
```
短线段聚类 → 在面板边缘附近 → 按几何关系组合：
  - 两条平行短线 + 一条连接线 → SLOT（U形）
  - 从边缘延伸的矩形 → TONGUE
  - TONGUE端部的反向短线 → HOOK
```

**对称性辅助**：锁扣通常成对出现，找到一侧可镜像生成另一侧。

### 2. 防撞翼片识别（FLAP）

**特征**：
- 45°斜线组，通常在面板角落
- 锯齿形排列（齿谷-齿尖交替）
- 深度 = H × 0.3~0.5

**识别方法**：
```
斜线聚类（30°-60°）→ 按位置分组 → 
  - 在面板角落 → FLAP候选
  - 齿形分析：测量齿间距、齿深
  - 对称性：左右翼片应镜像
```

### 3. 圆弧识别

**特征**：
- 曲率突变点（R通常3-20mm）
- 常见于：插舌端部、锁扣圆角、耳朵圆弧

**识别方法**：
```
轮廓曲率分析 → 
  - θ'(s)/θ''(s) 微分
  - 连续高曲率区域 → 弧段
  - 圆拟合 → R + 圆心
```

### 4. 耳朵（插舌/防尘翼）

**特征**：
- 从面板边缘延伸的梯形/三角形
- 底边在面板边上，顶边较窄
- FEFCO标准：W_tongue = (t + C_res) × K_stru
