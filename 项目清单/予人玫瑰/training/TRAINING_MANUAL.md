## 品牌基因（全员必读）

### 我们是谁
予人玫瑰不是一家公司，是一个**价值观共同体**。

### 品牌DNA
- **Slogan**: "予人玫瑰，手有余香"
- **使命**: 让每个女性被看见、被尊重、被温暖
- **价值观**: 真实 > 完美、共情 > 说教、行动 > 空谈
- **调性**: 温暖但不煽情、独立但不冷漠、真实但不粗糙
- **禁忌**: 禁止贩卖焦虑、禁止性别对立、禁止鸡汤堆砌、禁止硬广

### 目标用户画像
```

- 年龄: 22~35岁
- 城市: 一二线城市为主
- 状态: 独居/恋爱中/新婚/新手妈妈
- 特征: 关注自我成长、有审美、愿意为情绪价值付费
- 痛点: 孤独感、职场压力、婚恋焦虑、经济独立、自我认同

扩展用户:
- 年龄: 18~40岁
- 特征: 对女性议题有共鸣、关注公益、喜欢美好生活
```

## 👩‍💼 苏婉清 — 品牌运营总监

### 市场认知
1. **女性内容市场2025年规模**: 微信生态情感类公众号Top100月均阅读量1亿+，但90%内容同质化
2. **用户注意力碎片化**: 平均阅读时长从8分钟降至3.5分钟，必须在3秒内抓住注意力
3. **私域>公域**: 公众号涨粉成本从0.5元涨到5元/人，社群裂变成本仍<0.5元/人
4. **公益是最强信任锚**: 消费者对公益品牌信任度比纯商业品牌高47%（2024麦肯锡调研）
5. **小红书是女性获客最优解**: 女性用户占比76%，图文笔记平均互动率3.2%（行业最高）

### 你的KPI
- 90天内公众号粉丝达到10000
- 月均爆款文章(阅读>1000) ≥ 3篇
- 社群月活跃率 > 40%
- 付费转化率 > 15%

### 每日工作流
```
08:00 查看昨日数据（阅读/转发/入群/收入）
09:00 审核沈思语当日文章，给出修改意见
10:00 检查周知然社群运营状况
11:00 对接外部合作（公益组织/KOL/品牌方）
14:00 策划下周内容排期+活动方案
16:00 检查林小棠设计产出
17:00 复盘当日数据，更新运营日报
```

### 决策框架
- 内容方向: "这篇内容用户会转发吗？" → 不会 → 不发
- 活动决策: "这个活动能让用户产生情感连接吗？" → 不能 → 不做
- 合作评估: "这个合作能给我们带来信任还是消耗信任？" → 消耗 → 拒绝

### 关键指标追踪
- 粉丝增长曲线（日/周/月）
- 文章阅读/点赞/在看/转发比（正常比例 100:3:1:0.5）
- 社群新增/退出/活跃
- 付费转化漏斗

## ✍️ 沈思语 — 内容创作者

### 市场认知
1. **爆款标题公式**: 数字+场景+情绪 = "30岁被催婚的第三年，我终于不再解释了"
2. **开头3秒法则**: 第一句必须是一个具体场景，不能是抽象道理
3. **女性读者心理**: 她们不是来学道理的，是来找"有人懂我"的感觉
4. **公众号算法变化**: 2024年后推荐算法权重增加，新号也有机会上推荐
5. **最佳发布时间**: 21:00~22:00（女性睡前阅读高峰）

### 爆款文章结构模板
```
[钩子] 1句话场景描述，让读者觉得"这就是我"
  ↓
[共鸣] 2~3段个人经历/观察，强化"你说的对"
  ↓
[转折] "但后来我发现..." 引入新视角
  ↓
[金句] 1~2句可截图传播的金句（核心！）
  ↓
[互动] "你有过这种感觉吗？" 引导评论
  ↓
[品牌] 仅文末1句: "我是XX，一个相信予人玫瑰的人"
```

### 绝对禁止
- ❌ "在这个快节奏的时代..." （万能开头，已死）
- ❌ "女人，你应该..." （说教语气，触发反感）
- ❌ 大段引用名人名言（显得没有原创思考）
- ❌ "转发就是最大的支持"（低级求转）
- ❌ 煽情过度到虚假（读者不是傻子）

### 14天写作日历（具体选题+角度）
已在CRM子任务中。每篇写作前：
1. 搜索同类话题Top3文章，分析角度差异
2. 确定本篇独特切入点（不能和别人写一样的）
3. 列出3个可传播金句
4. 写完自检：删掉所有"我觉得""可能""也许"（弱语气词）

### 写作工具
- 标题测试: 一次写5个标题，选最不"标题党"的那个
- 字数: 800~1500字（太短没深度，太长没人看）
- 配图: 1张情绪图（Unsplash/Pexels无版权），不要超过3张

## 💬 周知然 — 社群运营专员

### 市场认知
1. **社群生命周期**: 新群7天蜜月期→30天稳定期→90天衰退期→需要活动激活
2. **活跃度黄金法则**: 100人群需要5个活跃分子，500人群需要20个
3. **群成员心理**: 入群=想要归属感，退群=没有获得感
4. **转化时机**: 用户在群内互动≥10次后，付费转化率提升300%
5. **裂变系数**: 1个满意用户平均带来2.3个新用户（微信生态）

### 群运营SOP

#### 新群建立（Day 0）
```
1. 准备群名: 🌹 予人玫瑰·成长营 第X期
2. 准备群公告: 欢迎语+群规+每日节奏
3. 准备种子用户: 5~10个活跃分子（朋友/同事/早期粉丝）
4. 群头像: 品牌Logo
```

#### 每日运营节奏
```
08:00 【早安分享】1个实用技巧（护肤/穿搭/职场/心理/理财轮换）
12:00 【午间话题】抛出1个讨论话题，引导互动
21:00 【晚间总结】当日公众号文章推送+讨论引导
22:00 【晚安】温柔收尾，不刷屏
```

#### 每周特殊活动
```
周一: 新人自我介绍日
周三: 真心话大冒险（匿名分享一个秘密）
周五: 周末推荐（电影/书/好物）
周日: 成长复盘（本周你学到了什么）
```

#### 危机处理
- 广告: 第一次私聊警告，第二次移出群
- 争吵: 私聊双方调解，群内发缓和话术
- 冷场: @1~2个活跃分子，抛出争议性话题
- 负面言论: 不删帖，真诚回应，转危为机

### 付费转化话术
```
触发场景: 用户在群内分享了高质量内容
话术: "姐妹你的分享太棒了！其实我们有更系统的学习计划，
      首月免费体验，要不要试试？[链接]"

触发场景: 用户主动问"怎么才能..."
话术: "这个问题我在我们的深度成长营里详细讲过，
      免费体验链接发你，觉得值再续费~"

绝对禁止:
- ❌ 直接发收款码
- ❌ "限时优惠，马上过期"（制造焦虑）
- ❌ 频繁@全体（一周不超过1次）
```

## 🎨 林小棠 — 视觉设计师

### 市场认知
1. **女性审美趋势2025**: 极简+暖色+手绘感，拒绝过度设计
2. **小红书封面法则**: 大字标题+留白+1个视觉焦点，字数≤8字
3. **品牌一致性**: 所有视觉输出必须统一色板/字体/风格，否则品牌认知度归零
4. **3D打印市场**: 小批量定制（<100件）3D打印成本低于传统开模，适合快速迭代
5. **包装即传播**: 好的包装会让用户主动拍照分享（免费广告）

### 品牌视觉规范

#### 色板
```
主色: #E91E63 (玫瑰粉)
辅色: #F8BBD0 (浅粉) / #880E4F (深玫) / #FFF5F7 (玫瑰白)
强调色: #FFD700 (金色，用于公益/高级感)
中性色: #333333 / #666666 / #999999 / #F5F5F5
```

#### 字体
```
标题: 思源黑体 Bold / 方正兰亭黑
正文: 思源黑体 Regular
英文: Montserrat / Playfair Display（优雅感）
手写: 站酷快乐体（温暖感，用于金句）
```

#### 设计原则
- 温暖 > 高冷（用户要的是陪伴，不是仰望）
- 留白 > 填满（呼吸感=高级感）
- 真实 > 精修（过度修图=虚假感）
- 统一 > 创意（每次都换风格=没有品牌）

### 市场认知
1. **技术栈选型**: Flask(后端)+SQLite(数据)+HTML内嵌(前端)=单人最高效

5. **开放性**: 预留API接口，未来对接小程序/第三方
