## 数据库扩展

现有表: customers, tasks, followups, feedbacks, users, settings

### 新增表
```sql
-- 公司信息
company_info (key, value, updated_at)

-- 项目/里程碑
projects (id, name, description, status, start_date, target_date, progress_pct)

-- 里程碑
milestones (id, project_id, title, target_date, status, completed_at)

-- 品牌资产
brand_assets (id, type, name, file_path, version, tags, status)

-- 内容管理
content_articles (id, title, content, platform, status, publish_date, read_count, share_count)

-- 供应商
suppliers (id, name, category, contact, phone, rating, notes)

-- 传播活动
campaigns (id, name, type, status, start_date, end_date, metrics_json)
```

## 当前优先执行顺序

1. **数据库扩展** — 新增表结构，兼容现有数据
2. **任务系统升级** — 子任务+模板+看板
3. **项目仪表盘** — 可视化进度
4. **公司注册/Logo模块** — 填充真实数据
5. **内容管理** — 公众号推文+排期
6. **Web UI** — Flask + 前端界面
