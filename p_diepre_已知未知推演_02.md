## 一、已知/未知划分

### 1.1 裱合复合结构的已知事实

**F22. 裱合工序存在三个耦合误差源（从节点数据涌现）**
- 置信度: ★★★★☆ (多节点交叉验证)
- 内容: 裱合工序的三个误差源为：
  1. 胶水固化收缩（拉力，约0.02-0.05%/干燥周期）
  2. 面纸吸湿膨胀（推力，CD方向0.015-0.028%/1%RH）
  3. 张力控制差异（裱合机进纸速度差导致）
- **关键发现**: 拉力与推力同时发生，不是简单相消。方向相同时叠加，方向相反时取决于哪个主导。
- RSS模型将其视为独立变量**可能低估耦合效应**，因为胶水固化过程会改变面纸局部的MC分布。

**F23. 灰板裱面纸的"喇叭口"变形（物理规律）**
- 置信度: ★★★★★ (力学可推导)
- 内容: 面纸纹理方向与灰板纹理垂直时，干燥后面纸CD方向收缩 vs 灰板MD方向不收缩 → 产生喇叭口
- 包边位要求: 面纸必须大于灰板 1.5-3mm（不可违反的物理约束）
- **推论**: 纹理方向匹配是精度的前置条件，不是优化项

**F24. 裱合胶水的"钉扎效应"（从节点数据涌现）**
- 置信度: ★★★☆☆ (单节点，需验证)
- 内容: 胶水渗透进芯纸后硬化，形成局部刚度增强点（钉扎点），导致弯曲回弹不均匀
- 影响: K因子在局部区域波动±0.05，但对整体展开长度影响有限（<0.1mm/300mm）
- **结论**: 可忽略，除非做亚毫米级精度

### 1.2 K因子分层体系

**F25. K因子精确值数据库（从1562节点中提取汇总）**

| 材料 | K值 | 中性轴偏移 | 数据来源 |
|------|-----|-----------|---------|
| 灰板(≤1.5mm) | 0.35-0.40 | 向内偏移15-20% | 江浙沪修正版+FEFCO |
| 灰板(>1.5mm) | 0.40 | 向内偏移20% | 力学推导 |
| 单瓦楞 A/B/C | 0.35 | 向内偏移15% | 标准值 |
| 单瓦楞 E/F | 0.40-0.45 | 向内偏移10-15% | 微瓦楞修正 |
| 双瓦楞 BC/AB | 0.40 | 向内偏移20% | 保守估计 |
| 高强瓦楞 | 0.38 | 向内偏移12% | 行业实测 |
| 压线过深(爆线临界) | 0.25 | 极端偏移 | 失效边界 |
| 塑料PVC | 0.50 | 中心对称 | 各向同性材料 |

**关键洞察**: K值随厚度增加而增大（0.35→0.40），但到达爆线临界时骤降到0.25。这不是连续函数，存在相变。

**F26. 双瓦楞的"外实内虚"风险**
- 置信度: ★★★★☆
- 内容: BC/AB坑在0.5T压线深度下，内层面纸承受剪切力→内层瓦楞塌陷
- **修正**: 双瓦楞推荐压线深度 0.45T（比单瓦楞浅10%）
- 展开公式: L = Σ(A_i) + Σ(B_i) + (n-1)×t×K_dual, K_dual = 0.40

### 1.3 微瓦楞盲区

**F27. 微瓦楞(E/F)不能直接套用FEFCO公式**
- 置信度: ★★★★★
- 内容: FEFCO公式 W=1.5t 适用于A/B/C粗瓦楞。微瓦楞(E:1.5mm, F:0.8mm)因压实系数不同，直接套用导致槽宽过宽
- **修正公式**: W = (t + C_res) × K_stru
  - E楞: C_res=0.5, K_stru=1.2 → W ≈ 1.2t + 0.6
  - F楞: C_res=0.3, K_stru=1.3 → W ≈ 1.3t + 0.4
- **压痕深度**: d = 0.4-0.45T（比粗瓦楞浅，防止塌陷）

**F28. JIS标准在微瓦楞领域的优势**
- 置信度: ★★★☆☆ (引用节点，未获取JIS原文)
- 内容: JIS Z 1506对微瓦楞折叠展开有具体k值规定
- 与FEFCO差异: 预计 ≥0.2mm（节点数据）
- **需人类确认**: 获取JIS Z 1506原文

## 三、新涌现的公式/规则

### R7. 复合结构展开长度通用公式

$$L = \sum A_i + \sum B_i + \sum_{j=1}^{n-1} t_j \times K_j + \Delta_{lamination}$$

其中:
- $A_i$: 各面板外尺寸
- $B_i$: 各面板外尺寸
- $t_j$: 第j个折弯处的总厚度
- $K_j$: 第j个折弯处的K因子（查F25表）
- $\Delta_{lamination}$: 裱合收缩补偿（V8待量化，暂取0）

### R8. 灰板裱面纸的纹理匹配规则

```
IF 面纸纹理 ∥ 灰板纹理 → 收缩率一致，无喇叭口 → Δ_lamination ≈ 0
IF 面纸纹理 ⊥ 灰板纹理 → 喇叭口变形 → 需预留面纸余量1.5-3mm
IF 不确定纹理方向 → 默认假设垂直（保守），加包边位
```
